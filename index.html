This is The 7 days 7 Websites Coding Challenge

This is one of some tricks to improve your coding skills. It's inspired from Florin Pop dev.to blog.
The main rule is simple: build one website, every day, for a week.
#7Days7Websites.

The official website of challenge is here.

The Rules
  1. Create a website every day for a week.
  2. You can use any programming language and/or any framework you want.
  3. The website can be on any topic.
  4. Submit your entries on the submit page.
  5. Share your progress on Social Media using the #7Days7Websites hashtag.
